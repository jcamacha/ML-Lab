### PASO 4: Crear servicios HTTP en src/app/services/

Cada servicio usa HttpClient. Ejemplo de api.service.ts:
```typescript
import { Injectable } from '@angular/core';
import { HttpClient, HttpHeaders, HttpErrorResponse } from '@angular/common/http';
import { Observable, throwError } from 'rxjs';
import { catchError } from 'rxjs/operators';
import { environment } from '../../environments/environment';

@Injectable({ providedIn: 'root' })
export class ApiService {
  protected apiUrl = environment.apiUrl;
  
  constructor(protected http: HttpClient) {}

  protected handleError(error: HttpErrorResponse) {
    console.error('API Error:', error);
    return throwError(() => new Error(error.message || 'Error del servidor'));
  }
}
```

Luego extiende ApiService para crear:
- **user.service.ts**: getAll(), getById(id), create(user), update(id, user), delete(id), login(email, password)
- **dataset.service.ts**: getAll(), getById(id), create(dataset), update(id, dataset), delete(id), getByType(type)
- **model.service.ts**: getAll(), getById(id), create(model), update(id, model), delete(id)
- **file.service.ts**: upload(formData: FormData), getAll(), getById(id), delete(id)
- **experiment.service.ts**: getAll(), getById(id), create(exp), getByUserId(uid), getByModelId(mid), run(expId)
- **ml.service.ts**: runLinear(data), runLogistic(data), runKnn(data) — POST al backend
- **email.service.ts**: sendEmail(email) — POST con Email
- **pdf.service.ts**: getExperimentPdf(id), getDatasetPdf(id) — responseType: 'blob'
