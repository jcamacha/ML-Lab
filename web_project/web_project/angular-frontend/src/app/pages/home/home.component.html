Sobre el Proyecto

Esta plataforma web permite a alumnos y profesores interactuar con conjuntos de datos, entrenar y evaluar modelos de Machine Learning, gestionar archivos en la nube y generar reportes PDF/UA estructurados. Diseñado como una herramienta integral para la experimentación práctica y rigurosa en el aula.
