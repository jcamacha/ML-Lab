Laboratorio de Machine Learning (ML Lab)

Plataforma educativa para la experimentación con algoritmos de aprendizaje automático.

Sobre el Proyecto

Esta aplicación web permite a alumnos y profesores de ESCOM interactuar con datasets, entrenar y evaluar modelos de Machine Learning (Regresión Lineal, Regresión Logística, KNN), gestionar archivos de datos y generar reportes en formato PDF. Es una herramienta diseñada para facilitar el aprendizaje práctico de la Inteligencia Artificial.
