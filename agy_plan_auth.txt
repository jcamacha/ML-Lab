## REQUERIMIENTOS

### 1. Sistema de Login/Signin
- Login con email + password
- Signin/registro de nuevos usuarios
- Persistencia de sesion (JWT o localStorage)
- Proteccion de rutas (AuthGuard)
- Navbar cambia cuando el usuario esta logueado (muestra nombre, boton logout)
- Login page estilizada con el tema premium existente

### 2. Admin
- Admin predefinido: email=juanESCOM@ipn.com, password=prueba
- Admin es responsable de funciones "peligrosas":
  * Eliminar usuarios
  * Eliminar experimentos
  * Eliminar datasets
  * Eliminar modelos
  * Ver todos los datos del sistema
- Usuarios regulares NO pueden borrar cosas de otros
- Admin ve un panel especial de administracion

### 3. Mejora UX
Basado en tu entendimiento de la app (laboratorio de ML para ejecutar experimentos):
- Dashboard con metricas relevantes
- Home page que explique el proposito de la app
- Flujo claro: registrarse -> crear dataset -> ejecutar experimento -> ver resultados
- Navegacion intuitiva
